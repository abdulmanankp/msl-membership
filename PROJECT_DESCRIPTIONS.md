# MSL Pakistan - Comprehensive Member Management System

## Overview

MSL Pakistan is a production-ready, full-stack member management system 
designed for organizations to efficiently manage member registrations, 
verify identities, generate personalized membership cards, and communicate 
with members via WhatsApp. Built with modern technologies and deployed 
using automated GitHub integration on Hostinger Node.js hosting.

## Key Features

### Member Management
- **Simple Online Registration**: User-friendly signup form with email verification
- **Profile Management**: Store and manage comprehensive member information
- **Membership Approval**: Admin approval workflow with automated WhatsApp + email notifications
- **Member Verification**: QR code-based member status verification system
- **Activity Tracking**: Monitor member registrations, downloads, and engagement

### WhatsApp Integration
- **Cloud API v17.0**: Official Facebook WhatsApp Cloud API integration
- **OTP Delivery**: Send one-time passwords for verification
- **Approval Notifications**: Automatic WhatsApp messages when membership is approved
- **Delivery Tracking**: Monitor message delivery and read receipts
- **Webhook Verification**: Proper webhook handling per Facebook documentation

### Email Notifications
- **SMTP Integration**: Hostinger email service support
- **Registration Emails**: Automated welcome emails upon signup
- **Approval Emails**: Detailed approval notifications with membership ID
- **Professional Templates**: HTML + plain text email templates
- **Error Alerts**: Administrator notifications for system errors

### PDF Card Generation
- **Custom Templates**: Upload and manage custom PDF templates
- **Dynamic Field Mapping**: Map form fields to PDF placeholders
- **Profile Integration**: Include member photos and biographical data
- **QR Codes**: Auto-generated QR codes for member verification
- **No Server Storage**: Cards generated and streamed directly to users

### Admin Controls
- **Centralized Dashboard**: Complete member and system management interface
- **Member Management**: Search, view, and manage all member records
- **WhatsApp Settings**: Enable/disable WhatsApp, manage business configuration
- **Download Limits**: Set configurable weekly per-member download quotas
- **Template Management**: Upload and configure PDF templates
- **System Monitoring**: View application logs and track system health

### System Monitoring
- **Error Logging**: File-based error tracking with ISO timestamps
- **Activity Logs**: Track all user and system activities
- **Email Delivery Logs**: Monitor email sending status
- **WhatsApp Event Logs**: Track message delivery and receipts

## Technology Stack

### Frontend
- React 18 with TypeScript for type-safe UI development
- Vite for lightning-fast development and optimized builds
- Tailwind CSS for modern utility-first styling
- shadcn/ui component library for professional UI elements
- Supabase JS client for database and authentication

### Backend
- Node.js 18+ runtime environment
- Express.js for robust API server
- Nodemailer for email delivery
- pdfme for PDF manipulation and generation
- File-based JSON storage for settings and tracking

### Services & Infrastructure
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **WhatsApp Messaging**: Facebook Cloud API v17.0
- **Email Service**: Hostinger SMTP
- **Hosting**: Hostinger Node.js hosting
- **Version Control**: GitHub with auto-deploy integration

## Deployment & DevOps

### Automated Deployment
- **GitHub Integration**: Connect repository for automatic deployment
- **Auto-Deploy**: Changes deploy within 2-3 minutes of pushing to main branch
- **Environment Management**: Secure environment variable management via hosting provider
- **Zero Downtime**: Seamless updates without service interruption

### Production Readiness
- Error logging and monitoring
- Database backups enabled
- HTTPS/SSL encryption
- Security best practices implemented
- Scalable architecture

## Getting Started

### For Developers
1. Clone GitHub repository
2. Install Node.js dependencies: `npm install`
3. Configure environment variables
4. Run development servers: `npm run dev` and `npm run server`
5. Make changes and push to GitHub
6. Hostinger auto-deploys your changes

### For Users
1. Visit platform website
2. Register membership account
3. Receive verification email
4. Admin reviews and approves
5. Download personalized membership card

- CORS protection
- Rate limiting ready
